# Road to AI 🚀

A practical, hands-on guide to learning Artificial Intelligence from the ground up. This repository contains Jupyter notebooks covering Python fundamentals, statistics, probability, and data analysis — the essential building blocks for any AI/ML engineer.

## Contents

### 📐 Statistics & Probability
| Notebook | Topics |
|---|---|
| `mean-median-mode.ipynb` | Central tendency measures, distributions |
| `standard-deviation-variance.ipynb` | Spread measures, normal distribution |

### 🐍 Python for Data Science
| Notebook | Topics |
|---|---|
| `basic.ipynb` | Python fundamentals for AI/ML |
| `pandas.ipynb` | Data manipulation, analysis, cleaning |

### 📊 Datasets
- `assets/PastHires.csv` — Sample dataset for hands-on practice

```bash

cd road-to-ai
pip install jupyter pandas numpy matplotlib
jupyter notebook
```
